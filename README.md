## Setup

Compile program:
```
mkdir build
cd build
cmake ..
make 
./PhonebookCLI
```

Run Program:
```
cd ./build/bin
./PhonebookCLI
```
Note: Jika CMakeLists.txt diubah maka hapus dahulu folder dengan `rm -rf build` lalu ulang dari awal proses di atas
